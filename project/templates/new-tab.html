Create new

You can create a graph sending data via curl 
curl -H "Content-Type: application/json" -d '{
    "name": "pings",
    "units": "ms"
}' https://{{ VIRTUAL_HOST }}/graph/

curl -d name=pings -d units=ms https://{{ VIRTUAL_HOST }}/graph/

{
    "graph-key": "{{ EXAMPLE_GRAPH_DOC_ID }}"
}

graph-key is your unique graph Identifier

Send values to existing graph

You can send other values to specified graph 
curl -H "Content-Type: application/json" -d '{
    "value": "10",
    "date": "{{ DATES[0] }}"
}' https://{{ VIRTUAL_HOST }}/graph/{{ EXAMPLE_GRAPH_DOC_ID }}/

curl -d value=10 -d date="2016-07-18T18:35:08.088%2B03:00" https://{{ VIRTUAL_HOST }}/graph/{{ EXAMPLE_GRAPH_DOC_ID }}/

curl -H "Content-Type: application/json" -d '{
    "value": "6",
    "unixtimestamp": 1468861804
}'  https://{{ VIRTUAL_HOST }}/graph/{{ EXAMPLE_GRAPH_DOC_ID }}/

curl -d value=6 -d unixtimestamp=1468861804 https://{{ VIRTUAL_HOST }}/graph/{{ EXAMPLE_GRAPH_DOC_ID }}/


curl -H "Content-Type: application/json" -d '{
    "value": "1"
}'  https://{{ VIRTUAL_HOST }}/graph/{{ EXAMPLE_GRAPH_DOC_ID }}/
